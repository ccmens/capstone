# Capstone project

## api 
1. Nodejs
2. Express
3. Mongodb


## web frontend 
1. React
2. Ant Design

## Backend Installation
1. cd api
2. npm install
3. npm run dev or npm run start
   1. some configuration (src/config.js)

## Frontend Installation
1. cd web
2. npm install
3. npm run start
4. npm run build (for production)
   1. src/Config.js should be changed `appBaseUrl`
